# Poeciliidae Gut Length

This repository contains the data and R code used in the manuscript:

**Ardón, D., Johnson, E. S., Johnson, E. E., Johnson, J. B.** *Gut length reflects trophic ecology but shows no tradeoff with reproductive allotment in livebearing fishes.*

## Repository contents

- **Poeciliid_specimen_data.csv** – Dataset used in all statistical analyses presented in the manuscript.
- **GutLength_Analysis.R** – R script reproducing the statistical analyses, figures, and supplementary tables.

## Notes

All analyses can be reproduced using the files provided in this repository.

Original gut photographs are not included because of their size but are available from the corresponding author upon reasonable request.
